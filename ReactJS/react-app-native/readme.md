* `npm install react-redux`
* `npm install --save-dev redux-devtools` > use com a extensão react dev tools do chrome
* `yarn add react-router-dom` > rotas

## Notas Pessoais:
* React:
  * React > Biblioteca != Angular > Framework
  * React Native
  * UWP > react de aplicativo windows (netflix usa)
  * componentes
  * SPA > single page application
 
* Redux:
  * Me parece um tipo de hook useState dentro de um contexto, ao invés de você modificar o valor você faz uma pedido para o valor ser modificado.
  * redux toolkit ajuda com padroes ``--template redux`` usa ele. nao precisa nem instalar o reduz com isto. Uitilize esse kit para usar o redux.
  * alternativas:
      * Contex API
      * Recoil 
      * MobX
      * XState 
 
* webpack:
  * entry > ponto de entrada, arquivo principal dos modulos
  * loaders > gerenciar arquivos como css, images, fontes, etc...
 
* HTTP Requests:
  * Fetch > Nova funcionalidade, navegadores mais antigos não o aceitam.
  * Axios > Utiliza XHTTPREQUEST que é antigo, tem mais compatibilidade, melhor monitoramento, tratamento de erros, teste, pode monitorar a request...
 
* ``debugger`` > Depuração > colocar direto no codigo e ve na hora a renderização? Tipo um breakpoint do C#?
